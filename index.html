Sobre mim

Dados organizados viram decisões melhores.

Atuo como estagiário em Ciência de Dados na Geap, com foco em análise, organização e otimização de bases de dados para apoiar a tomada de decisão. Sou formado em Análise e Desenvolvimento de Sistemas pela Universidade Católica de Brasília e planejo iniciar em breve uma pós-graduação para aprofundar ainda mais minhas competências voltadas para o trabalho com dados.

No estágio, uso Excel avançado e Power BI no dia a dia para estruturar informações, cruzar bases e construir dashboards e indicadores. Em projetos pessoais, aprofundo Python (Pandas, NumPy, Matplotlib), SQL e Machine Learning — sempre buscando transformar volumes grandes de dados em algo simples de entender.
